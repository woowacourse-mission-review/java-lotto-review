# java-lotto
로또 미션 진행을 위한 저장소

## 기능 목록

- 구입 금액 입력 받기
```text
- 로또 1장의 가격은 1000원
- 공백 제거
- 정수 입력이 아닌 경우 예외 처리
- 1000원 미만의 가격인 경우 예외 처리
```

- 수동 구매 로또 수 입력 받기
```text
- 공백 제거
- 정수 입력이 아닌 경우 예외 처리
- 음수 입력인 경우 예외 처리
- 0인 경우 자동으로 점프
```

- 수동으로 구매할 로또 입력 받기
```text
- 숫자는 쉼표(,)로 구분
- 공백 제거
- 숫자가 아닌 경우 예외 처리
- 숫자가 6개가 아닌 경우 예외 처리
- 숫자가 로또(1~45) 숫자의 범위가 아닌 경우 예외 처리
- 중복된 숫자가 있을 경우 예외 처리
```

- 수동, 자동 로또 구매 번호 출력하기


- 지난 주 당첨 번호 입력 받기
```text
- 숫자는 쉼표(,)로 구분
- 공백 제거
- 숫자가 아닌 경우 예외 처리
- 숫자가 6개가 아닌 경우 예외 처리
- 숫자가 로또(1~45) 숫자의 범위가 아닌 경우 예외 처리
- 중복된 숫자가 있을 경우 예외 처리
```

- 보너스 볼 입력 받기
```text
- 공백 제거
- 숫자가 아닌 경우 예외 처리
- 숫자가 로또(1~45) 숫자의 범위가 아닌 경우 예외 처리
- 지난 주 당첨 번호와 중복되는 경우 예외 처리
```


- 당첨 통계 출력하기
